### Information Hierarchy for Events
1. Event name (most prominent): fontSize 56-80, fontWeight "bold", center of poster
2. Date & time: fontSize 24-32, positioned prominently below the title
3. Venue / location: fontSize 20-28, near the date
4. Featured performers / speakers: fontSize 22-30 each
5. Call to action ("Get Tickets", "Register Now"): rect button + text, high contrast
6. Additional details (sponsors, website, price): fontSize 12-16, bottom of poster

### Date Formatting
- Stylized large date: separate text elements for day (fontSize 72+) and month (fontSize 24)
- Calendar block: rect with rounded corners containing date elements
- Place date in upper-right or directly below event name

### Location Display
- Text with venue name at fontSize 20-24
- Address below at fontSize 14-16, opacity 0.7
- Use a small decorative line above or below to separate from other info

### Speaker/Performer Cards
- Rect card background with rounded corners (rx: 12, ry: 12)
- Name text (fontWeight "bold", fontSize 20-24)
- Title/role text (fontSize 14-16, opacity 0.7)
- Arrange in a 2-3 column layout using calculated left positions

### Schedule/Agenda
- Left-aligned time entries: text elements with fontWeight "bold"
- Right-aligned session names with regular fontWeight
- Horizontal line dividers between entries
- Use consistent vertical spacing (24-32px between rows)

### CTA Buttons
- Rect background with rounded corners (rx: 24-30 for pill shape)
- Text element centered on the rect
- High contrast: bright fill rect (#3b82f6, #f59e0b) with white text
- Add urgency labels as smaller text above: "Limited Seats", "Early Bird"
- Position prominently in the lower third of the poster
